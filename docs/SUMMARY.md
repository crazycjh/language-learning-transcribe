## 注意事項

1. **環境設置**
   - 確保 Node.js >= 18.0.0
   - 設置必要的環境變數
   - 配置 R2 存取權限

2. **開發流程**
   - 遵循 TypeScript 類型定義
   - 使用提供的組件和服務
   - 遵守API規範

3. **最佳實踐**
   - 使用提供的類型定義
   - 實現錯誤處理
   - 遵循性能優化建議
